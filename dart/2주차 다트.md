# 인터페이스
dart에서는 인터페이스를 class로 선언해 사용한다.
구헌 클래스에서 인터페이스를 implements 하면된다.
```dart
class WorldInterface {
  String? name;
  void sayName(){}
}

class Korea implements WorldInterface{
  String? name;
  Korea(
    String? name,
  ):this.name= name;
  
  void sayName(){
    print('이 나라의 이름은 ${this.name}입니다.');
  }
}

class Usa implements WorldInterface {
  String? name;
  
  Usa(
    String? name,
  ):this.name = name;
  
  void sayName(){
    print('이 나라의 이름은 ${this.name}입니다.');
  }
}

void main() {
  Korea korea = new Korea('한국');
  korea.sayName();
  Usa usa = new Usa('미국');
  usa.sayName();
}
```
> 이 나라의 이름은 한국입니다.<br>
> 이 나라의 이름은 미국입니다.
