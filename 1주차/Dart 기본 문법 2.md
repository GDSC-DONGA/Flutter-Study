### 다트의 상속

자식 클래스는 extends 키워드 통해 상속 받고자 하는 부모 클래스 지정
@override는 부모 클래스의 메서드 재정의 하고 싶을 때 사용
> 재정의하다 : 기존 메서드에서 구현한 내용 대신 다른 동작을 하는 코드 구현
```
class 부모 클래스명{
  멤버 변수;
  멈버 함수(){
  }
}

class 자식 클래스 명 extends 부모 클래스명{
  @override
  멤버 함수(){
  }
}
```
