# 1. 데이터 타입
## int
정수형

555, 1, 3, ...

int a = 555;

## double
실수형

0.0001 ... 

double b = 11.1;

## String
문자열
스트링의 맨앞은 S 대문자입니다.

String str1 = "BlockDMask";

String str2 = 'BlockDMask';

큰따옴표 작은따옴표 둘다 사용가능이지만 작은따옴표가 표준이라고 함.

## bool
참 거짓을 나타내는 불리언 타입

true, false

bool c = false;

## num
못보던 타입.

int와 double을 포함하는 타입 입니다.

num n1 = 44;

num n2 = 23.12;

int, double로 선언한 변수를 num 타입에 담을 수 있습니다.

int a = 10;

num n = a; //가능
